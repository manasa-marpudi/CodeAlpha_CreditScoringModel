## Project Title

## Objective
The main objective of this project is to predict whether a person is creditworthy or not using Machine Learning algorithms based on financial history and customer-related information.

---

## Dataset Used
The project uses a credit scoring dataset containing financial details such as:

- Income
- Debt
- Payment History
- Loan Amount
- Age
